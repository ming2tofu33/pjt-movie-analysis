# 🎥🎬 한국 영화 데이터 분석 프로젝트 (2005-2019)

## 🍿 프로젝트 소개

이 프로젝트는 **2005년부터 2019년까지의 한국 영화 데이터**를 수집, 전처리, 분석 및 시각화하는 것을 목표로 합니다.

주요 분석 주제는 **장르별 트렌드, 흥행 요인, 평점과 흥행의 상관관계, 배우·감독 네트워크 분석** 등입니다.

**분석 기간을 2019년으로 한정한 이유**: 코로나19 팬데믹 이후 영화 시장의 왜곡된 특수 상황을 배제하고, 2000년대 중반부터 본격적으로 자리 잡은 한류 확산기를 반영하여 한국 영화 산업의 온전한 흐름을 포착하기 위함입니다.

---

## 🔄 데이터 수집 여정과 시행착오

### 1차 시도: TMDB API

- **목표**: 글로벌 영화 데이터베이스를 활용한 한국 영화 분석
- **한계**:
    - 한국 영화 데이터 표본 부족 (약 87편)
    - 예산·흥행 수익 정보 미비
    - 한국 영화 평점 데이터 부족

### 2차 시도: IMDB 평점 데이터 보완

- **목표**: TMDB 데이터의 평점 부족 문제 해결
- **시도**: IMDB에서 한국 영화 평점 데이터 수집
- **결과**: 한국 영화 데이터가 사실상 없어서 실패

### 3차 시도: TMDB Overview 키워드 분석

- **시도**: TMDB의 영화 개요(overview) 텍스트에서 키워드 추출
- **방법**: 흥행 영화의 개요에서 특정 키워드(가족, 전쟁, 사랑) 빈도 분석
- **결과**: 최종 데이터셋에는 이런 텍스트 데이터가 없어 재현 불가

### 최종 해결책: KOFIC + 네이버 영화

- **KOFIC API**: 공식 박스오피스 데이터, 관객 수, 수익 데이터
- **네이버 영화 크롤링**: 네티즌 평점, 비평가 평점
- **수동 입력**: 크롤링 불가능한 약 200편 직접 입력으로 품질 보강
- **결과**: **927편의 완성된 데이터셋 구축**

## 📊 데이터 개요

**최종 데이터셋 규모**: **927편 × 23개 컬럼**

**데이터 출처**:

- KOFIC API (영화진흥위원회): 박스오피스, 관객 수, 수익
- 네이버 영화: 네티즌 평점, 비평가 평점
- 수동 입력: 크롤링 불가 데이터 보완

**수집 기간**: 2005-01-01 ~ 2019-12-31

**수집 기준**:

- KOFIC 주간 박스오피스 Top 10에 한 번이라도 등장한 영화
- 한국 제작 영화만 필터링

**주요 컬럼**:

- `movie_id`: 영화 고유 식별자 (KOFIC 기준)
- `title`: 영화 제목
- `release_date`: 개봉일
- `genres`: 장르
- `runtime`: 상영 시간
- `revenue`: 총 수익 (원)
- `audience_total`: 총 관객 수
- `vote_average_naver`: 네이버 네티즌 평점
- `vote_count_naver`: 네이버 네티즌 평가 수
- `critic_average`: 네이버 비평가 평점
- `critic_count`: 네이버 비평가 수
- `director_name_ko`: 감독 이름 (한국어)
- `cast_names_ko`: 주요 배우 이름 (한국어, | 구분)
- `production_companies`: 제작사
- `release_year`, `release_month`: 개봉 연도, 월

## 🎯 주요 발견사항 미리보기

### 🎭 사극의 특별함

- **한국 영화만의 독특한 흥행 장르**
- 평균 수익 **226억원** (전체 장르 중 1위)
- 조선시대 역사·실존 인물 기반 서사가 대중적 흡인력과 문화적 정체성 제공
- 대표작: 《명량》, 《광해》, 《관상》, 《사도》

### 📊 평점 vs 흥행의 괴리

- 네티즌 평점-흥행 수익: **상관계수 0.22** (약한 상관관계)
- 비평가 평점-흥행 수익: **상관계수 0.19** (약한 상관관계)
- **"평점이 낮아도 규모의 힘(자본+배급+마케팅)으로 흥행 가능"**

### 🎬 네트워크 효과

- **감독-배우 협업이 흥행에 미치는 구조적 영향**
- 주요 성공 콤보:
    - 김용화-하정우/주지훈: 평균 수익 1,091억
    - 최동훈-이정재/전지현: 959억
    - 봉준호-송강호: 736억

### 📅 계절별 개봉 전략

- **공포**: 여름 집중 개봉 → ROI 극대화
- **코미디·판타지**: 겨울·연말 → 가족 단위 수요 집중
- **드라마**: 사계절 고른 분포 → 안정적 수익 구조

## ⚠️ 한계점 및 제약사항

### 데이터 관련

- **예산 데이터 부재**: KOFIC에서 제작비 정보를 제공하지 않아 ROI 분석 불가능
- **표본 편향**: 박스오피스 Top 10 진입 영화만 수집 (소규모 독립영화 제외)
- **해외 수익 제외**: 국내 수익만 집계되어 글로벌 성과 미반영

### 산업적 한계

- **산업 투명성 부족**: 한국 영화 산업의 제작비·투자 내역 불투명성
- **구조적 리스크**: 박스오피스 데이터는 공개되지만 투자 구조는 불투명

### 🔑 1. API 키 설정

이 프로젝트에서는 KOFIC API를 사용합니다.

1. [KOFIC 오픈API](https://www.kobis.or.kr/kobisopenapi/homepg/main/main.do) 웹사이트에서 회원가입 및 로그인

## 🔎 분석 과제

### 과제 1: 장르별 영화 트렌드 및 흥행 성공 요인

- 장르별 분포 및 수익 분석
- 개봉 시기와 장르의 상관관계
- 사극의 특수성 분석

### 과제 2: 영화 평점과 흥행 수익 간의 상관관계

- 네티즌 vs 비평가 평점 비교
- 평점-흥행 괴리 현상 분석
- 저평점·고흥행 vs 고평점·저흥행 그룹 특성 분석

- 감독-배우 협업 네트워크 구조 분석
- 성공적인 협업 패턴 발견
- 네트워크 중심성과 흥행의 관계

### 과제 4: 종합 분석 및 인사이트 도출

- 흥행 성공 요인 종합 분석
- 한국 영화 산업의 특성 및 시사점
- 향후 연구 방향 제시
